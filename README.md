# Sprite Fighter — Goblin Enemies

## Sprites
- Put your Adventurer sheet as: `assets/player.png`
- Put the Goblin sheet as: `assets/goblin.png`

Goblin grid assumed: 12 columns x 6 rows.

Rows used:
- walk: row 1
- sword attack: row 3 (used as hit reaction)
- death: row 5

If these are different, tell me and I'll adjust.
