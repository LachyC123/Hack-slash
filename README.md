# Sprite Fighter — On-screen Joystick (iPhone)

## 1) Put your sprite sheet
Replace:
`assets/character.png`
with your original itch.io PNG (350x407).

## 2) Upload to GitHub
Upload all files and keep the `assets` folder.

## 3) Turn on GitHub Pages
Settings → Pages → Deploy from branch → main / root.

## Controls
- Mobile: joystick = move, buttons = punch/slide
- Desktop: WASD/arrows, J punch, Shift slide
